DORG

A custom computing ecosystem

  • DCPU is a custom 8-bit CPU that combines old-school simplicity with modern ambition.
  • It has its own ISA and instruction encoding.
  • Its goal is to be easy to understand and learn at all levels.
  • Even the machine code is designed to be relatively easy to read.
  • Built from scratch to learn how CPUs work.
  • DASM is the official assembler for DCPU.
  • Supports all instructions and modifiers.
  • Data directives allow inserting raw data into RAM.
  • Includes labels and comments for readable assembly code.
  • A UTF-8 text display designed for DCPU.
  • Capable of printing characters and clearing the screen.
  • A simple programming language designed for DCPU.
  • Inspired by C while respecting DASM constraints.
  • Supports 3 data types.
  • Includes loops (while / for), if-else statements, functions, and pointers.
  • Features a preprocessor for libraries.
