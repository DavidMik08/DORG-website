DCPU
  • DCPU is a custom 8 bit CPU that combines old-school simplicity and modern ambition.
  • It has its own ISA and instruction encoding.
  • Its goal is to be easy to understand and learn at all levels,
    even the machine code isn't too hard to learn.
  • It was built from scratch by Petar Filo and David Mikacic
    because we wanted to learn more about how CPUs work.
DASM
  • DASM is the official assembler for DCPU.
  • It supports all of the instructions and modifiers.
  • It has data directives
    that allow you to put some data into ram with the instructions.
  • It also supporrts comments and labels.
DCPU text screen
  • DCPU text screen is an UTF-8 text based screen
    made for the DCPU
  • It can display characters and clear the screen
DLANG
  • DLANG is a simple language made for DCPU that is inspired by c,
    but it's made with DASM constrictions in mind.
  • It has 3 data types, while and for loops, if-else statements, functions, pointers
    and a preprocesor so you can include libraries.
